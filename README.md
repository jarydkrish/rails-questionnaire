## Steps to get started
1. Click the "Deploy to Heroku" button
2. Open the application
3. Go to the URL `/users/sign_up`
4. Create a user/password
5. Go to `/admin/questions`
6. Add a question
7. Go to `/questions`
8. Add an answer
